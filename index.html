Tampilan & Format Gaji

Integrasi Database (Google Sheets)

1
Siapkan Spreadsheet Kosong

Buat file Google Spreadsheet baru di Drive Anda. Script pintar ini dilengkapi fitur Dynamic Headers sehingga kolom akan digenerate otomatis. Bebas modifikasi urutan!

2
Pasang & Izinkan Script

Buka menu Extensions > Apps Script, hapus kode yang ada, lalu *paste* script di bawah. Pastikan untuk menekan tombol Run (Jalankan) fungsi "setup" pertama kali dan berikan izin ke akun Google Anda.

3
Deploy & Tautkan Sistem

Pilih Deploy > New deployment (Web app). Atur akses ke "Anyone". Salin URL yang dihasilkan dan tempel ke kolom pengaman di bawah ini.
