Pengaturan Aplikasi & Database

Pengaturan Tampilan & Perhitungan

Perubahan akan langsung diterapkan pada seluruh data di aplikasi.

Langkah 1: Siapkan Spreadsheet

Buat Google Spreadsheet baru. Anda tidak perlu membuat header manual. Script pintar ini akan otomatis membuat header dan bisa membaca data walaupun urutan kolomnya diacak (Dynamic Headers)!

Langkah 2: Pasang Apps Script

Klik menu Extensions > Apps Script di spreadsheet Anda, hapus semua kode di sana, dan tempel kode di bawah ini.

SANGAT PENTING: Sebelum di-Deploy!

Pilih fungsi setup pada dropdown di atas editor, lalu klik tombol Run (Jalankan). Google akan meminta izin akses ke Google Drive Anda. Wajib diizinkan agar gambar tidak hilang dan bisa tersimpan!

Langkah 3: Deploy & Dapatkan URL

Klik tombol Deploy > New deployment. Pilih tipe Web app. Atur "Execute as" ke "Me" dan "Who has access" ke Anyone. Salin URL Web app yang dihasilkan.
